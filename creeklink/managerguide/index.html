EDIT CREEK SEGMENTS


Edit creek segments by clicking in the "Creek Segments" area to select it (grey it out).

EDIT CREEK SEGMENTS

Close

  • Choose Edit an existing feature by clicking the "Edit existing feature" icon.
  • Click on the feature you would like to modify.
  • Click the edit button on the bottom left of the emerging panel.
  • Move the anchor points in order to change the shape, direction or length of the feature and enter data in the required fields.
  • Click on the tickable element row or press space bar to activate or deactivate that option.
  • Activate the 'visited' option to change the colour of the creek segment to indicate that it has been visited.
  • Turn on the "Creek Segments Visited" layer to display visited creek segments.

CREATE CREEK SEGMENTS


Create creek segments by clicking in the "Creek Segments" area to select it (grey it out).

CREATE CREEK SEGMENTS

Close

  • Create a new feature by clicking the "Create a new feature" icon.
  • Click once on the map where you would like your feature to begin.
  • Click once again to change the direction of the connected line.
  • Change direction as many times as you need or double click to end the feature. (Note that intermediate anchors appear whenever an anchor is moved for the first time. These anchors, although initially dimmed, have all the functionality of the original anchors).
  • A panel now emerges where you can enter the data in the required fields.
  • Click save to save your work.

EDIT LANDCARE BOUNDARY


Edit landcare boundary by ticking the "Landcare Boundary (for edit)" checkbox and clicking in the "Landcare Boundary (for edit)" area to select it (grey it out).

EDIT LANDCARE BOUNDARY

Close

  • Edit an existing landcare boundary by clicking on the "Edit existing feature" icon.
  • Click on the boundary you would like to modify.
  • Click the edit button on the bottom left of the emerging panel.
  • Move the anchor points in order to change the shape or size of the group.
  • You can also rename the landcare group name.
