CREATE CREEK SEGMENTS


Create creek segments by clicking in the "Creek Segments" area to select it (grey it out).

CREATE CREEK SEGMENTS

Close

If you choose the "Create a new feature" icon you can create a new feature by clicking once on the map where you would like your feature to begin then click once again to change the direction of the connected line. You may change direction as many times as you need or double click to end the feature. Note that intermediate anchors appear whenever an anchor is moved for the first time. These anchors, although initially dimmed, have all the functionality of the original anchors.

A panel now emerges where you can enter the data in the required fields. Then click save to save your work.

EDIT LANDCARE BOUNDARY


Edit landcare boundary by ticking the "Landcare Boundary (for edit)" checkbox and clicking in the "Landcare Boundary (for edit)" area to select it (grey it out).

EDIT LANDCARE BOUNDARY

Close

If you choose the "Edit existing feature" icon you can edit an existing landcare boundary by clicking on the boundary you would like to modify. You then click the edit button on the bottom left of the emerging panel. You can then move the anchor points in order to change the shape or size of the group. You can also rename the landcare group name.
